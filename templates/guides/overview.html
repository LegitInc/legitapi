Overview of the API

The Legit API lets Legit partners obtain identity risk and reputation information from Legit. It also allows them to submit reputation data that they have collected from their users.

All API methods are provided as web services.


Querying Data

Identity risk and reputation data is made available through the query methods. This information is used to verify and assess the risk of presented identities.

Read more about querying data.


Submitting Data

The submit methods let partners submit reputation data to Legit. Submitted data is combined with data sent in by other partners. The resulting aggregate reputation strengthens the data made available through the query methods.

Read more about submitting data.


Authentication

Authentication for all methods is done using OAuth 1.0a. When you signed up for a developer account you were assigned OAuth credentials for accessing the API. These can be found on your profile page.

Read more about authenticating.
